一、数据结构类型定义

队伍的信息定义为Team类，成员变量为

private:
string teamID;//参赛队编号
string workTitles;//参赛作品名称
string schoolName;//参赛学校
string category;// 赛事类别
string participant;//参赛者
string teacher;//指导老师

并给出读取与修改的成员函数public:
``public: void PrintTeamINFO(); void setTeamID(string teamID); void setWorkTitles(string workTitle); void setSchoolName(string schoolName); void setCategory(string category); void setParticipant(string participant); void setTeacher(string teacher); string getTeamID(); string getWorkTitles(); string getSchoolName(); string getCategory(); string getParticipant(); string getTeacher();`

由于添加时是从文件读取，不存在数据缺失，因此为了方便创建队伍，构造函数的参数为全体成员变量
